# OpenCapture

*OpenCapture* is a simple app to capture text and append it to a file on the local filesystem.

## Usage

1. Clone the repository: `git clone https://github.com/floscha/opencapture.git`
2. Install dependencies: `npm install`
3. Run the app: `npm run dev`
